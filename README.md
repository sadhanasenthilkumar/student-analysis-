Student Management System

Objective

To develop a console-based Student Management System in C++ that efficiently manages student records using file handling and menu-driven operations.

Features

- Add Student
- Display Student Records
- Search Student
- Update Student Information
- Delete Student Records
- Persistent Data Storage using File Handling

Technologies Used

- C++
- File Handling
- VS Code
- GCC Compiler

How to Run

Compile:
g++ main.cpp -o student

Run:
.\student.exe

Expected Outcome

The application allows users to manage student records efficiently with permanent storage in a text file.
